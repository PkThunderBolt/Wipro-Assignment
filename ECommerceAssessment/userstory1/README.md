# User Story 1 – Product Showcase Home Page

## Scenario

This project is part of an E-Commerce Product Portal.  
The goal is to design a responsive Home Page that displays featured products, includes category navigation, and allows users to subscribe to a newsletter.

---

## Objective

Build a Product Showcase Home Page using semantic HTML5 and Bootstrap, featuring:

- A category navigation bar
- Product cards with images, names, and prices
- A newsletter subscription form with validation
- Responsive layout and hover transitions

---

## Features Implemented

-Semantic HTML5 tags – header, nav, main, section, and footer  
 -Bootstrap grid system – ensures full responsiveness  
 -Product cards – showcase image, product name, and price  
 -Category navigation bar – easy product category browsing  
 -Newsletter subscription form – includes basic validation  
 -CSS hover effects – smooth scaling and shadow transitions  
 -Sticky footer – always remains at the bottom of the page

## Page Overview

### 1 Header & Navigation

- Displays brand name and navigation links (Home, Categories, Contact, etc.)

### 2 Featured Products Section

- Products are displayed using Bootstrap cards
- Each card includes an image, name, and price
- On hover, the card slightly scales up for a modern UI effect

### 3 Newsletter Subscription Form

- Simple form for users to enter their email
- Includes HTML5 validation to ensure proper email format

### 4 Footer

- Fixed to the bottom
- Contains copyright
